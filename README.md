# ­ЪЄг­ЪЄф ESP-IDF-рЃўрЃА рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ ­ЪЄг­ЪЄф

рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ! ­ЪЉІ

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА [Espressif-рЃўрЃА ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/index.html) рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃАрЃарЃБрЃџ, рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃА рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ. ­ЪЊќ

## ­Ъј» рЃарЃљ рЃљрЃарЃўрЃА ESP-IDF?

ESP-IDF (Espressif IoT Development Framework) рЃљрЃарЃўрЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ Espressif-рЃўрЃА рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЏрЃўрЃЎрЃарЃЮрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ ESP32, ESP32-S2, ESP32-C3 рЃЊрЃљ рЃАрЃ«рЃЋрЃљ. рЃўрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃА, рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃА, рЃарЃљрЃЌрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў - рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃЮрЃфрЃўрЃЏрЃфрЃўрЃЏрЃћ LED-рЃЊрЃљрЃю рЃЊрЃљрЃгрЃДрЃћрЃЉрЃБрЃџрЃў, рЃарЃЌрЃБрЃџрЃў IoT рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃЏрЃЌрЃљрЃЋрЃарЃћрЃЉрЃБрЃџрЃў. ­ЪњА

## ­ЪДљ рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃарЃўрЃА рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ?

рЃћрЃА рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃў рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљрЃАрЃЌрЃЋрЃўрЃА, рЃЋрЃўрЃАрЃљрЃф рЃАрЃБрЃарЃА ESP-IDF-рЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ:

-   **рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА:** ­ЪЉЕРђЇ­Ъњ╗ рЃЌрЃБ рЃљрЃ«рЃџрЃљ рЃўрЃгрЃДрЃћрЃЉрЃЌ рЃЏрЃўрЃЎрЃарЃЮрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
-   **рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:** ­ЪјЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃџрЃљрЃЉрЃЮрЃарЃљрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
-   **рЃ░рЃЮрЃЉрЃўрЃАрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:** ­ЪЏа№ИЈ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃљрЃАрЃљрЃгрЃДрЃЮрЃЉрЃљрЃЊ.
-   **рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:** ­ЪДЉРђЇ­ЪћД рЃЎрЃЮрЃЏрЃћрЃарЃфрЃўрЃБрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ.

## ­Ъџђ рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮ?

рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ! рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЕрЃЋрЃћрЃюрЃА рЃАрЃљрЃўрЃбрЃќрЃћ рЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ. рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ рЃАрЃћрЃЦрЃфрЃўрЃћрЃЉрЃљрЃЊ, рЃарЃљрЃф рЃњрЃљрЃњрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃЌрЃћрЃЏрЃўрЃА рЃърЃЮрЃЋрЃюрЃљрЃА.
