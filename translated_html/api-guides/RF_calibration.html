რადიოსიხშირული კალიბრაცია¶

ESP32 მხარს უჭერს RF ინიციალიზაციის დროს სამ RF კალიბრაციის მეთოდს:

  1. ნაწილობრივი კალიბრაცია
  2. სრული კალიბრაცია
  3. კალიბრაცია არ არის

ნაწილობრივი კალიბრაცია¶

RF ინიციალიზაციის დროს, RF კალიბრაციისთვის ნაგულისხმევად გამოიყენება ნაწილობრივი კალიბრაციის მეთოდი. ის ხორციელდება NVS-ში შენახული სრული კალიბრაციის მონაცემების საფუძველზე. ამ მეთოდის გამოსაყენებლად, გთხოვთ, გადახვიდეთ menuconfig და ჩართეთ CONFIG_ESP32_PHY_CALIBRATION_AND_DATA_STORAGE.

სრული კალიბრაცია¶

სრული კალიბრაცია ხორციელდება შემდეგ პირობებში:

  1. NVS არ არსებობს.
  2. კალიბრაციის მონაცემების შესანახი NVS დანაყოფი წაშლილია.
  3. აპარატურის MAC მისამართი შეიცვალა.
  4. PHY ბიბლიოთეკის ვერსია შეიცვალა.
  5. NVS დანაყოფიდან ჩატვირთული RF კალიბრაციის მონაცემები დაზიანებულია.

ნაწილობრივ კალიბრაციასთან შედარებით, დაახლოებით 100 მილიწამზე მეტი დრო სჭირდება. თუ ჩატვირთვის ხანგრძლივობა კრიტიკული არ არის, რეკომენდებულია სრული კალიბრაციის მეთოდის გამოყენება. სრულ კალიბრაციის მეთოდზე გადასასვლელად, გადადით menuconfig და გამორთვა CONFIG_ESP32_PHY_CALIBRATION_AND_DATA_STORAGEთუ იყენებთ RF კალიბრაციის ნაგულისხმევ მეთოდს, სრული კალიბრაციის გააქტიურების ფუნქციის, როგორც უკიდურესი საშუალების დამატების ორი გზა არსებობს.

  1. თუ არ გაწუხებთ, წაშალეთ NVS დანაყოფი და მასში შენახული ყველა მონაცემი წაიშლება. ეს მართლაც ყველაზე მარტივი გზაა.
  2. დარეკეთ API esp_phy_erase_cal_data_in_nvs() WiFi-სა და BT/BLE-ს ინიციალიზაციამდე გარკვეული პირობების საფუძველზე (მაგ., დიაგნოსტიკური რეჟიმით გათვალისწინებული ვარიანტი). ამ შემთხვევაში, მხოლოდ NVS დანაყოფის ფიზიკური სახელთა სივრცე წაიშლება.

კალიბრაცია არ არის¶

კალიბრაციის მეთოდი არ გამოიყენება მხოლოდ მაშინ, როდესაც ESP32 ღრმა ძილიდან იღვიძებს.

PHY ინიციალიზაციის მონაცემები¶

RF კალიბრაციისთვის გამოიყენება PHY ინიციალიზაციის მონაცემები. PHY ინიციალიზაციის მონაცემების მისაღებად ორი გზა არსებობს.

ერთი არის ნაგულისხმევი ინიციალიზაციის მონაცემები, რომელიც მდებარეობს სათაურის ფაილში. კომპონენტი s/esp_wifi/esp32/include/phy_init_data.hკომპილაციის შემდეგ ის ჩაშენებულია აპლიკაციის ბინარულ ფაილში და შემდეგ ინახება მხოლოდ წასაკითხ მეხსიერებაში (DROM). ნაგულისხმევი ინიციალიზაციის მონაცემების გამოსაყენებლად, გთხოვთ, გადახვიდეთ menuconfig და გამორთვა CONFIG_ESP32_PHY_INIT_DATA_IN_PARTITION.

კიდევ ერთი არის ინიციალიზაციის მონაცემები, რომლებიც ინახება დანაყოფში. მორგებული დანაყოფების ცხრილი გამოყენებისას დარწმუნდით, რომ PHY მონაცემთა დანაყოფი შედის (ტიპი: data, ქვეტიპი: phy). ნაგულისხმევი დანაყოფების ცხრილი ით, ეს ავტომატურად ხდება. თუ ინიციალიზაციის მონაცემები ინახება დანაყოფში, ის იქ უნდა იყოს ფლეშ მეხსიერება ის რედაქტირებული, წინააღმდეგ შემთხვევაში მოხდება გაშვების შეცდომა. დანაყოფში შენახულ ინიციალიზაციის მონაცემებზე გადასასვლელად, გადადით menuconfig და ჩართეთ CONFIG_ESP32_PHY_INIT_DATA_IN_PARTITION.
