მხარდაჭერილი ფუნქციები¶

ბადისებრი ბირთვი¶

  • Provisioning: Node Role
    • PB-ADV და PB-GATT
    • OOB ავთენტიფიკაცია
  • Provisioning: Provisioner Role
    • PB-ADV და PB-GATT
    • OOB ავთენტიფიკაცია
  • Networking
    • ესტაფეტა
    • სეგმენტაცია და ხელახლა აწყობა
    • გასაღების განახლების პროცედურა
    • IV განახლების პროცედურა
    • მეგობარი
    • დაბალი სიმძლავრე
    • პროქსი სერვერი
    • პროქსი კლიენტი
  • Multiple Client Models Run Simultaneously
    • მხარი დაუჭირეთ მრავალი კლიენტის მოდელს, რომლებიც ერთდროულად აგზავნიან პაკეტებს სხვადასხვა კვანძებში
    • კლიენტისა და სერვერის მოდელებს შორის ბლოკირება არ არის
  • NVS Storing
    • ESP-BLE-MESH კვანძის უზრუნველყოფისა და კონფიგურაციის ინფორმაციის შენახვა
