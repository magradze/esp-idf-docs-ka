Eclipse IDE-ს გამოყენებით აწყობა და ფლეშ-პროგრამირება¶

[中文]

ESP-IDF V4.0-ს ნაგულისხმევ აწყობის სისტემად ახალი CMake-ზე დაფუძნებული აწყობის სისტემა აქვს.

არსებობს ახალი ESP-IDF Eclipse მოდული, რომელიც მუშაობს CMake-ზე დაფუძნებულ აწყობის სისტემასთან. გთხოვთ, იხილოთ https://github.com/espressif/idf-eclipse-plugin/blob/master/README.md დამატებითი ინსტრუქციებისთვის.

თუ Eclipse IDE-ს მხარდაჭერა გჭირდებათ მემკვიდრეობით მიღებული ESP_IDF Make build სისტემისთვის, შეგიძლიათ მიჰყვეთ შემდეგ ინსტრუქციას: მემკვიდრეობით მიღებული GNU შექმნის სისტემის შექმნის სახელმძღვანელო რომელსაც აქვს ნაბიჯები Eclipse IDE-ს გამოყენებით აწყობა და ფლეშირება.
