ინსტალაციის წინაპირობები¶

კომპილაცია ისთვის ESP-IDF ით თქვენ უნდა მიიღოთ შემდეგი პაკეტები:

  • CentOS 7:

    sudo yum install git wget flex bison gperf python pyserial python-pyelftools cmake ninja-build ccache

  • Ubuntu და Debian:

    sudo apt-get install git wget flex bison gperf python python-pip python-setuptools python-serial python-click python-cryptography python-future python-pyparsing python-pyelftools cmake ninja-build ccache libffi-dev libssl-dev

  • თაღი:

    sudo pacman -S --needed gcc git make flex bison gperf python2-pip python2-pyserial python2-click python2-cryptography python2-future python2-pyparsing python2-pyelftools cmake ninja ccache

შენიშვნა

ESP-IDF თან გამოსაყენებლად საჭიროა CMake ვერსია 3.5 ან უფრო ახალი. Linux-ის ძველ დისტრიბუციებს შეიძლება დასჭირდეთ განახლება, „backports“ რეპოზიტორის ჩართვა ან „cmake“-ის ნაცვლად „cmake3“ პაკეტის ინსტალაცია.

დამატებითი რჩევები¶

ნებართვის პრობლემები /dev/ttyUSB0¶

ზოგიერთი Linux დისტრიბუციის შემთხვევაში, შეიძლება მიიღოთ Failed to open port /dev/ttyUSB0 შეცდომის შეტყობინება ფლეშ მეხსიერება ESP32-ის დაყენებისას. ამის მოგვარება შესაძლებელია მიმდინარე მომხმარებლის dialout ჯგუფში დამატებით..

შემდეგი ნაბიჯები¶
