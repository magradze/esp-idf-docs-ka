Toolchain-ის სტანდარტული დაყენება Mac OS-ისთვის¶

[中文]

ინსტალაციის წინაპირობები¶

ESP-IDF გამოიყენებს Python-ის ვერსიას, რომელიც სტანდარტულად არის დაინსტალირებული macOS-ზე.

  • პიპის ინსტალაცია:

    sudo easy_install pip

  • დააინსტალირეთ pyserial:

    pip install --user pyserial

  • დააინსტალირეთ CMake &amp; Ninja ბილდი:

    • თუ გაქვთ HomeBrew, შეგიძლიათ გაუშვათ:

      brew install cmake ninja

    • თუ გაქვთ მაკპორტები, შეგიძლიათ გაუშვათ:

      sudo port install cmake ninja

    • წინააღმდეგ შემთხვევაში, მიმართეთ CMake და ნინძა macOS ინსტალაციის ჩამოტვირთვის მთავარი გვერდები.

  • ასევე მკაცრად რეკომენდებულია ინსტალაცია ტკივილი უფრო სწრაფი აწყობისთვის. თუ გაქვთ HomeBrew, ეს შეიძლება გაკეთდეს მეშვეობით brew install ccache ან sudo port install ccache ჩართულია მაკპორტები.

შენიშვნა

თუ რომელიმე ნაბიჯის დროს მსგავსი შეცდომა გამოჩნდება:

xcrun: error: invalid active developer path (/Library/Developer/CommandLineTools), missing xcrun at: /Library/Developer/CommandLineTools/usr/bin/xcrun

შემდეგ გასაგრძელებლად დაგჭირდებათ XCode ბრძანების ხაზის ინსტრუმენტების ინსტალაცია. მათი ინსტალაცია შეგიძლიათ ბრძანების გაშვებით xcode-select --install.
