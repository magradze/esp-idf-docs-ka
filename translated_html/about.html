შესახებ¶

[中文]

ეს არის დოკუმენტაცია, ESP-IDF, ჩარჩო აპლიკაციების შესამუშავებლად ESP32 ჩიპით ესპრესო.

ESP32 არის 2.4 GHz Wi-Fi და Bluetooth კომბინაცია, რომელიც აერთიანებს ერთ ან ორ 32-ბიტიან მიკროპროცესორს, 600 DMIPS-მდე დამუშავების სიმძლავრით.

Espressif IoT ინტეგრირებული განვითარების ჩარჩო

ESP-IDF , Espressif IoT Development Framework, უზრუნველყოფს ინსტრუმენტების ნაკრები , API , კომპონენტი ინსტრუმენტებსა და სამუშაო პროცესებს ESP32-ისთვის აპლიკაციების შესამუშავებლად Windows, Linux და Mac OS ოპერაციული სისტემების გამოყენებით.
