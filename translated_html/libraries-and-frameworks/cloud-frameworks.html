ღრუბლოვანი ჩარჩოები¶

ESP32 მხარს უჭერს რამდენიმე ღრუბლოვან ჩარჩო ს ESP-IDF ზე აგებული აგენტების გამოყენებით. აქ მოცემულია სხვადასხვა მხარდაჭერილი ღრუბლოვანი ჩარჩო აგენტების მითითებები და მაგალითები:

AWS ნივთების ინტერნეტი¶

https://github.com/espressif/esp-aws-iot არის ESP32-ის ღია კოდის საცავი, რომელიც დაფუძნებულია Amazon Web Services-ზე. aws-iot-device-sdk-embedded-C.

Azure IoT¶

https://github.com/espressif/esp-azure არის ESP32-ის ღია კოდის საცავი, რომელიც დაფუძნებულია Microsoft Azure-ის ბაზაზე. azure-iot-sdk-c SDK.

Google IoT Core¶

https://github.com/espressif/esp-google-iot არის ESP32-ის ღია კოდის საცავი, რომელიც დაფუძნებულია Google-ის ვერსიაზე. iot-device-sdk-embedded-c SDK.
