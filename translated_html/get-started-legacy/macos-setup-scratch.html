Mac OS-ისთვის ნულიდან დაყენების ინსტრუმენტების ჯაჭვი (Legacy GNU Make)¶

[中文]

შენიშვნა

ESP-IDF V4.0-დან დაწყებული, ნაგულისხმევი აწყობის სისტემა დაფუძნებულია CMake-ზე. ეს დოკუმენტაცია განკუთვნილია GNU Make-ზე დაფუძნებული მემკვიდრეობითი აწყობის სისტემისთვის. ამ აწყობის სისტემის მხარდაჭერა შესაძლოა მომავალ ძირითად ვერსიებში გაუქმდეს.

შენიშვნა

აღწერილია ინსტრუმენტების ნაკრები ინსტალაციის სტანდარტული პროცესი აქ. იხილეთ Toolchain-ის მორგებული დაყენება განყოფილებაში მოცემულია რამდენიმე მიზეზი, თუ რატომ შეიძლება საჭირო გახდეს ინსტრუმენტების ნაკრები ნულიდან ინსტალაცია.

ინსტალაციის წინაპირობები¶

  • პიპის ინსტალაცია:

    sudo easy_install pip

შენიშვნა

pip მოგვიანებით ინსტალაციისთვის გამოყენებული იქნება საჭირო Python პაკეტები.

წყაროდან ინსტრუმენტების ჯაჭვის კომპილაცია¶

  • ინსტალაციის დამოკიდებულებები:

    • დააინსტალირეთ რომელიმე მაკპორტები ან სახლში დამზადებული ლუდი პაკეტების მენეჯერი. MacPorts-ს სჭირდება XCode-ის სრული ინსტალაცია, ხოლო homebrew-ს მხოლოდ XCode ბრძანების ხაზის ინსტრუმენტები სჭირდება.

    • MacPorts-ით:

      sudo port install gsed gawk binutils gperf grep gettext wget libtool autoconf automake

    • სახლში დამზადებული ლუდით:

      brew install gnu-sed gawk binutils gperftools gettext wget help2man libtool autoconf automake

შექმენით ფაილური სისტემის გამოსახულება, რომელიც რეგისტრის მიმართ მგრძნობიარე იქნება:

hdiutil create ~/esp/crosstool.dmg -volname "ctng" -size 10g -fs "Case-sensitive HFS+"

დაამონტაჟეთ:

hdiutil mount ~/esp/crosstool.dmg

შექმენით სიმბოლური ბმული თქვენს სამუშაო დირექტორიაში:

mkdir -p ~/esp
ln -s /Volumes/ctng ~/esp/ctng-volume

გადადით ახლად შექმნილ დირექტორიაში:

cd ~/esp/ctng-volume

ჩამოტვირთვა crosstool-NG და ააშენეთ იგი:

git clone https://github.com/espressif/crosstool-NG.git
cd crosstool-NG
git checkout esp-2019r2
git submodule update --init
./bootstrap && ./configure --enable-local && make

შექმენით ინსტრუმენტების ნაკრები :

./ct-ng xtensa-esp32-elf
./ct-ng build
chmod -R u+w builds/xtensa-esp32-elf

ხელსაწყოების ჯაჭვი აშენდება ~/esp/ctng-volume/crosstool-NG/builds/xtensa-esp32-elf. გაჰყევით სტანდარტული დაყენების ინსტრუქციები რომ დაამატოთ ინსტრუმენტების ნაკრები თქვენს PATH.

შემდეგი ნაბიჯები¶
