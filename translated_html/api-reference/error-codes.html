შეცდომის კოდების მითითება¶

ამ განყოფილებაში ჩამოთვლილია ESP-IDF ში განსაზღვრული სხვადასხვა შეცდომის კოდის მუდმივები.

ESP-IDF -ში შეცდომის კოდების შესახებ ზოგადი ინფორმაციისთვის იხილეთ შეცდომების დამუშავება.

ESP_FAIL (-1): ზოგადი esp_err_t კოდი, რომელიც მიუთითებს შეცდომაზე

ESP_OK (0): esp_err_t მნიშვნელობა, რომელიც მიუთითებს წარმატებაზე (შეცდომა არ არის)

ESP_ERR_NVS_INVALID_LENGTH (0x110c)სტრიქონის ან blob-ის სიგრძე მონაცემების შესანახად საკმარისი არ არის.

ESP_ERR_NVS_NO_FREE_PAGES (0x110d)NVS დანაყოფი არ შეიცავს ცარიელ გვერდებს. ეს შეიძლება მოხდეს, თუ NVS დანაყოფი შემოკლებულია. წაშალეთ მთელი დანაყოფი და ხელახლა გამოიძახეთ nvs_ ფლეშ მეხსიერება _init.
