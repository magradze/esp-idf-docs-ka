ESP32-S2 ტემპერატურის სენსორი¶

მიმოხილვა¶

ESP32-S2-ს აქვს ჩაშენებული ტემპერატურის სენსორი. ტემპერატურის სენსორის მოდული შეიცავს 8-ბიტიან სიგმა-დელტა ანალოგურ-ციფრულ გადამყვანს და ტემპერატურის ოფსეტის ციფრულ-ციფრულ გადამყვანს. კონვერტაციის ურთიერთობა ქვემოთ მოცემული ცხრილის პირველი სვეტებია. მათ შორის, offset = 0 არის გაზომვის მთავარი ვარიანტი, ხოლო სხვა მნიშვნელობები არის გაფართოებული გაზომვის ვარიანტები.

ოფსეტი გაზომვის დიაპაზონი (ცელსიუსი) გაზომვის შეცდომა (ცელსიუსი)
-2 50 ~ 125 &lt; 3
-1 20 ~ 100 &lt; 2
0 -10 ~ 80 &lt; 1
1 -30 ~ 50 &lt; 2
2 -40 ~ 20 &lt; 3

განაცხადის მაგალითი¶

ტემპერატურის სენსორის წაკითხვის მაგალითი: პერიფერიული მოწყობილობები/ტემპერატურის_სენსორი.

esp_err_t temp_sensor_read_celsius(float *ცელსიუსი)¶

შენიშვნა
Should not be called from interrupt.

TSENS_DAC_L0 = 0¶

ოფსეტი = -2, გაზომვის დიაპაზონი: 50℃ ~ 125℃, შეცდომა &lt; 3℃.

TSENS_DAC_L1¶

ოფსეტი = -1, გაზომვის დიაპაზონი: 20℃ ~ 100℃, შეცდომა &lt; 2℃.

TSENS_DAC_L2¶

ოფსეტი = 0, გაზომვის დიაპაზონი: -10℃ ~ 80℃, შეცდომა &lt; 1℃.

TSENS_DAC_L3¶

ოფსეტი = 1, გაზომვის დიაპაზონი: -30℃ ~ 50℃, შეცდომა &lt; 2℃.

TSENS_DAC_L4¶

ოფსეტი = 2, გაზომვის დიაპაზონი: -40℃ ~ 20℃, შეცდომა &lt; 3℃.
