სიგმა-დელტა მოდულაცია¶

შესავალი¶

ESP32-ს აქვს მეორე რიგის სიგმა-დელტა მოდულაციის მოდული. ეს დრაივერი ახდენს სიგმა-დელტა მოდულის არხების კონფიგურაციას.

ფუნქციონალურობის მიმოხილვა¶

არსებობს რვა დამოუკიდებელი სიგმა-დელტა მოდულაციის არხი, რომლებიც იდენტიფიცირებულია sigmadelta_channel_tთითოეულ არხს შეუძლია სიგმა-დელტა მოდულაციით გამოსცეს ორობითი, აპარატურულად გენერირებული სიგნალი.

არჩეული არხი უნდა დაყენდეს კონფიგურაციის პარამეტრების მითითებით sigmadelta_config_t და შემდეგ ამ კონფიგურაციის გამოყენება sigmadelta_config().

კიდევ ერთი ვარიანტია ცალკეული ფუნქციების გამოძახება, რომლებიც ყველა საჭირო პარამეტრს ერთმანეთის მიყოლებით დააკონფიგურირებენ:

  • წინასწარი მასშტაბირება სიგმა-დელტა გენერატორის - sigmadelta_set_prescale()
  • მოვალეობა გამომავალი სიგნალის - sigmadelta_set_duty()
  • GPIO პინი მოდულირებული სიგნალის გამოსატანად - sigmadelta_set_pin()

„მოვალეობის“ შეყვანის პარამეტრის დიაპაზონი sigmadelta_set_duty() არის -128-დან 127-მდე (რვაბიტიანი ნიშანდებული მთელი რიცხვი). თუ დაყენებულია ნულოვანი მნიშვნელობა, მაშინ გამომავალი სიგნალის დატვირთვა იქნება დაახლოებით 50%, იხილეთ აღწერა sigmadelta_set_duty().

განაცხადის მაგალითი¶

სიგმა-დელტა მოდულაციის მაგალითი: პერიფერიული მოწყობილობები/სიგმადელტა.

API მითითება¶

სათაურის ფაილი¶

  • დრაივერი/ინკლუზია/დრაივერი/sigmadelta.h

ფუნქციები¶

esp_err_t sigmadelta_config(კონსტ sigmadelta_config_t *კონფიგურაცია)¶

სიგმა-დელტა არხის კონფიგურაცია.

დაბრუნება
  • ESP_OK წარმატება
  • ESP_ERR_INVALID_STATE სიგმადელტას დრაივერი უკვე ინიციალიზებულია
  • ESP_ERR_INVALID_ARG პარამეტრის შეცდომა
პარამეტრები
  • configსიგმა-დელტას არხის კონფიგურაციის სტრუქტურის მაჩვენებელი

esp_err_t sigmadelta_set_duty(sigmadelta_channel_t არხი, int8_t მოვალეობა)¶

სიგმა-დელტას არხის დავალების დაყენება.

ეს ფუნქცია გამოიყენება სიგმა-დელტა არხის დატვირთვის დასაყენებლად. თუ გამომავალ პინსა და მიწას შორის კონდენსატორს დაამატებთ, საშუალო გამომავალი ძაბვა იქნება Vdc = VDDIO / 256 * დატვირთვის + VDDIO/2, სადაც VDDIO არის კვების წყაროს ძაბვა.

დაბრუნება
  • ESP_OK წარმატება
  • ESP_ERR_INVALID_STATE sigmadelta დრაივერი არ არის ინიციალიზებული
  • ESP_ERR_INVALID_ARG პარამეტრის შეცდომა
პარამეტრები
  • channelსიგმა-დელტას არხის ნომერი
  • dutyერთი არხის სიგმა-დელტას დატვირთვა, მნიშვნელობა მერყეობს -128-დან 127-მდე, რეკომენდებული დიაპაზონია -90 ~ 90. ამ დიაპაზონში ტალღის ფორმა უფრო შემთხვევითია.

esp_err_t sigmadelta_set_prescale(sigmadelta_channel_t არხი, uint8_t წინასწარი მასშტაბირება)¶

დააყენეთ სიგმა-დელტა არხის ტაქტორის წინასწარი მასშტაბირების მნიშვნელობა. წყაროს ტაქტორია APP_CLK, 80 MHz. სიგმა-დელტა არხის ტაქტორის სიხშირეა APP_CLK / წინასწარი_მასშტაბი.

დაბრუნება
  • ESP_OK წარმატება
  • ESP_ERR_INVALID_STATE sigmadelta დრაივერი არ არის ინიციალიზებული
  • ESP_ERR_INVALID_ARG პარამეტრის შეცდომა
პარამეტრები
  • channelსიგმა-დელტას არხის ნომერი
  • prescaleწყაროს საათის გამყოფი, 0-დან 255-მდე დიაპაზონში

esp_err_t sigmadelta_set_pin(sigmadelta_channel_t არხი, gpio_num_t gpio_num)¶

დააყენეთ სიგმა-დელტას სიგნალის გამომავალი პინი.

დაბრუნება
  • ESP_OK წარმატება
  • ESP_ERR_INVALID_STATE sigmadelta დრაივერი არ არის ინიციალიზებული
  • ESP_ERR_INVALID_ARG პარამეტრის შეცდომა
პარამეტრები
  • channelსიგმა-დელტას არხის ნომერი
  • gpio_num: GPIO გამომავალი პინის ნომერი.
