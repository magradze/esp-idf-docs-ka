შეცდომის კოდები და დამხმარე ფუნქციები¶

ამ განყოფილებაში ჩამოთვლილია ტერმინი ESP-IDF შეცდომის კოდების განმარტებები და შეცდომების დამუშავებასთან დაკავშირებული რამდენიმე დამხმარე ფუნქცია.

ESP-IDF -ში შეცდომის კოდების შესახებ ზოგადი ინფორმაციისთვის იხილეთ შეცდომების დამუშავება.

ESP-IDF -ში განსაზღვრული შეცდომის კოდების სრული სიისთვის იხილეთ შეცდომის კოდის მითითება.

ფუნქციები¶

კონსტ char *esp_err_to_name(esp_err_t კოდი)¶

აბრუნებს სტრიქონს esp_err_t შეცდომის კოდებისთვის.

ეს ფუნქცია წინასწარ გენერირებულ საძიებო ცხრილში პოულობს შეცდომის კოდს და აბრუნებს მის სტრიქონულ წარმოდგენას.

ფუნქცია გენერირდება Python სკრიპტის ინსტრუმენტებით/gen_esp_err_to_name.py, რომელიც უნდა შესრულდეს ყოველ ჯერზე, როდესაც esp_err_t შეცდომა შეიცვლება, იქმნება ან წაიშლება IDF პროექტიდან.

დაბრუნება
string error message
პარამეტრები
  • code: esp_err_t შეცდომის კოდი

კონსტ char *esp_err_to_name_r(esp_err_t კოდი, char *ბუფი, size_t ბუფლენი)¶

აბრუნებს სტრიქონს esp_err_t-ისთვის და სისტემის შეცდომის კოდებისთვის.

ეს ფუნქცია შეცდომის კოდს პოულობს esp_err_t შეცდომების წინასწარ გენერირებულ საძიებო ცხრილში და აბრუნებს მის სტრიქონულ წარმოდგენას. თუ შეცდომის კოდი ვერ მოიძებნა, მაშინ მისი პოვნა სისტემის შეცდომებს შორის ხდება.

ფუნქცია გენერირდება Python სკრიპტის ინსტრუმენტებით/gen_esp_err_to_name.py, რომელიც უნდა შესრულდეს ყოველ ჯერზე, როდესაც esp_err_t შეცდომა შეიცვლება, იქმნება ან წაიშლება IDF პროექტიდან.

დაბრუნება
buf containing the string error message
პარამეტრები
  • code: esp_err_t შეცდომის კოდი
  • buf: ბუფერი, სადაც შეცდომის შეტყობინება უნდა დაიწეროს
  • buflenბუფერული ბუფერის ზომა. ბუფერულ ბუფერში იწერება მაქსიმუმ ბუფლენის ბაიტები (დამამთავრებელი ნულოვანი ბაიტის ჩათვლით).

მაკროები¶

ESP_OK¶

esp_err_t მნიშვნელობა, რომელიც მიუთითებს წარმატებაზე (შეცდომა არ არის)

ESP_FAIL¶

ზოგადი esp_err_t კოდი, რომელიც მიუთითებს წარუმატებლობაზე

ESP_ERR_NO_MEM¶

მეხსიერება ამოწურულია

ESP_ERR_INVALID_ARG¶

არასწორი არგუმენტი

ESP_ERR_INVALID_STATE¶

არასწორი შტატი

ESP_ERR_INVALID_SIZE¶

არასწორი ზომა

ESP_ERR_NOT_FOUND¶

მოთხოვნილი რესურსი ვერ მოიძებნა

ESP_ERR_NOT_SUPPORTED¶

ოპერაცია ან ფუნქცია არ არის მხარდაჭერილი

ESP_ERR_TIMEOUT¶

ოპერაციის დრო ამოიწურა

ESP_ERR_INVALID_RESPONSE¶

მიღებული პასუხი არასწორი იყო

ESP_ERR_INVALID_CRC¶

CRC ან ჩეკის ჯამი არასწორი იყო

ESP_ERR_INVALID_VERSION¶

ვერსია არასწორი იყო

ESP_ERR_INVALID_MAC¶

MAC მისამართი არასწორი იყო

ESP_ERR_WIFI_BASE¶

WiFi-ის შეცდომის კოდების საწყისი რაოდენობა

ESP_ERR_MESH_BASE¶

MESH შეცდომის კოდების საწყისი რაოდენობა

ESP_ERR_FLASH_BASE¶

ფლეშ მეხსიერება შეცდომის კოდების საწყისი ნომერი

ESP_ERROR_CHECK(x)¶

მაკრო, რომლის გამოყენებაც შესაძლებელია შეცდომის კოდის შესამოწმებლად და პროგრამის შეწყვეტისთვის, თუ კოდი არ არის ESP_OK. სერიულ გამოსავალზე ბეჭდავს შეცდომის კოდს, შეცდომის ადგილმდებარეობას და წარუმატებელი ოპერატორს.

გამორთულია, თუ დადასტურებები გამორთულია.

ESP_ERROR_CHECK_WITHOUT_ABORT(x)¶

მაკრო, რომლის გამოყენებაც შესაძლებელია შეცდომის კოდის შესამოწმებლად. სერიულ გამოსავალზე ბეჭდავს შეცდომის კოდს, შეცდომის ადგილმდებარეობას და წარუმატებელი ოპერატორის მონაცემებს. ESP_ERROR_CHECK()-თან შედარებით, ეს იგივე შეცდომის შეტყობინებას ბეჭდავს, მაგრამ პროგრამას არ წყვეტს.
