esp_err_t esp_smartconfig_start(კონსტ smartconfig_start_config_t *კონფიგურაცია)¶

გაუშვით SmartConfig, დააკონფიგურირეთ ESP მოწყობილობა წვდომის წერტილის დასაკავშირებლად. თქვენ უნდა გადასცეთ ინფორმაცია ტელეფონის აპლიკაციით. მოწყობილობა იჭერს სპეციალურ პაკეტებს ჰაერიდან, რომლებიც შეიცავს სამიზნე წვდომის წერტილის SSID-ს და პაროლს.

ყურადღება
1. This API can be called in station or softAP-station mode.
ყურადღება
2. Can not call esp_smartconfig_start twice before it finish, please call esp_smartconfig_stop first.
დაბრუნება
  • ESP_OK: წარმატება
  • სხვები: წარუმატებლობა
პარამეტრები
  • config: smartconfig-ის სტრუქტურის კონფიგურაციის დაწყების მაჩვენებელი

esp_err_t esp_smartconfig_stop(void)¶

შეაჩერეთ SmartConfig, გაათავისუფლეთ esp_smartconfig_start-ის მიერ დაკავებული ბუფერი.

ყურადღება
Whether connect to AP succeed or not, this API should be called to free memory taken by smartconfig_start.
დაბრუნება
  • ESP_OK: წარმატება
  • სხვები: წარუმატებლობა

esp_err_t esp_esptouch_set_timeout(uint8_t time_s)¶

SmartConfig პროცესის ვადის ამოწურვის დაყენება.

ყურადღება
Timing starts from SC_STATUS_FIND_CHANNEL status. SmartConfig will restart if timeout.
დაბრუნება
  • ESP_OK: წარმატება
  • სხვები: წარუმატებლობა
პარამეტრები
  • time_s: დიაპაზონი 15 წმ~255 წმ, ოფსეტი: 45 წმ.

esp_err_t esp_smartconfig_set_type(smartconfig_type_t ტიპი)¶

SmartConfig-ის პროტოკოლის ტიპის დაყენება.

ყურადღება
If users need to set the SmartConfig type, please set it before calling esp_smartconfig_start.
დაბრუნება
  • ESP_OK: წარმატება
  • სხვები: წარუმატებლობა
პარამეტრები
  • type: აირჩიეთ smartconfig_type_t-დან.

esp_err_t esp_smartconfig_fast_mode(bool ჩართვა)¶

SmartConfig-ის რეჟიმის დაყენება. ნაგულისხმევი ნორმალური რეჟიმი.

ყურადღება
1. Please call it before API esp_smartconfig_start.
ყურადღება
2. Fast mode have corresponding APP(phone).
ყურადღება
3. Two mode is compatible.
დაბრუნება
  • ESP_OK: წარმატება
  • სხვები: წარუმატებლობა
პარამეტრები
  • enable: false-disable(ნაგულისხმევი); true-enable;

uint8_t ssid[32]¶

წვდომის წერტილის SSID. ნულით დამთავრებული სტრიქონი.

uint8_t password[64]¶

წვდომის წერტილის პაროლი. ნულით შეწყვეტილი სტრიქონი.

bool bssid_set¶

დაყენებულია თუ არა სამიზნე წვდომის წერტილის MAC მისამართი.

uint8_t bssid[6]¶

სამიზნე წვდომის წერტილის MAC მისამართი.

smartconfig_type_t type¶

ჭკვიანი კონფიგურაციის ტიპი (ESPTouch ან AirKiss).

uint8_t token¶

მობილური ტელეფონის ტოკენი, რომელიც გამოიყენება ACK-ის გასაგზავნად მობილურ ტელეფონზე.

uint8_t cellphone_ip[4]¶

მობილური ტელეფონის IP მისამართი.

ჩამოთვლა smartconfig_event_t¶

Smartconfig-ის მოვლენების დეკლარაციები

ღირებულებები:

SC_EVENT_SCAN_DONE¶

ESP32 სადგურის smartconfig-მა დაასრულა წვდომის წერტილების სკანირება.

SC_EVENT_FOUND_CHANNEL¶

ESP32 სადგურის smartconfig-მა იპოვა სამიზნე წვდომის წერტილის არხი.

SC_EVENT_GOT_SSID_PSWD¶

ESP32 სადგურის smartconfig-მა მიიღო SSID და პაროლი.

SC_EVENT_SEND_ACK_DONE¶

ESP32 სადგურმა smartconfig-მა მობილურ ტელეფონზე ACK გაგზავნა
