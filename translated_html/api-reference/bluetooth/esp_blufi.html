BLUFI API¶

მიმოხილვა¶

BLUFI არის პროფილზე დაფუძნებული GATT, რომელიც გამოიყენება ESP32 WIFI-ის კონფიგურაციისთვის წვდომის წერტილის დასაკავშირებლად/გათიშვისთვის ან პროგრამული უზრუნველყოფის დასაყენებლად და ა.შ. გამოყენება უნდა ეხებოდეს შემდეგს:

  1. მოვლენა გაიგზავნა პროფილიდან. შემდეგ თქვენ უნდა გააკეთოთ ის, რაც მოვლენა მიუთითებს.
  2. უსაფრთხოების მითითება. თქვენ შეგიძლიათ დაწეროთ თქვენი საკუთარი უსაფრთხოების ფუნქციები, როგორიცაა სიმეტრიული დაშიფვრა/გაშიფვრა და ჩეკის ჯამის ფუნქციები. ასევე შეგიძლიათ განსაზღვროთ „გასაღების გაცვლის/მოლაპარაკების“ პროცედურა.

განაცხადის მაგალითი¶

შემოწმება ბლუთუზი/ბლუდროიდი/ბლე ESP-IDF examples-ში არსებული საქაღალდე, რომელიც შეიცავს შემდეგ აპლიკაციას:

  • ეს არის BLUFI-ის დემო ვერსია. ამ დემოს საშუალებით შეგიძლიათ ESP32-ის wifi-ის softap/station/softap&amp;station რეჟიმში დაყენება და wifi კავშირების კონფიგურაცია - ბლუთუზი/ბლუდროიდი/ბლე/ბლუფი
