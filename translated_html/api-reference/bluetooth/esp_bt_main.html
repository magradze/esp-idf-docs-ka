დაბრუნება
Bluetooth stack status

esp_err_t esp_bluedroid_enable(void)¶

esp_bluedroid_init()-ის შემდეგ აუცილებელია Bluetooth-ის ჩართვა

დაბრუნება
  • ESP_OK: წარმატებით დასრულდა
  • სხვა: ვერ მოხერხდა

esp_err_t esp_bluedroid_disable(void)¶

esp_bluedroid_deinit()-მდე აუცილებელია Bluetooth-ის გამორთვა.

დაბრუნება
  • ESP_OK: წარმატებით დასრულდა
  • სხვა: ვერ მოხერხდა

esp_err_t esp_bluedroid_init(void)¶

Bluetooth-ისთვის რესურსის ინიციალიზაცია და გამოყოფა უნდა მოხდეს ყველა Bluetooth-ის მოქმედებამდე.

დაბრუნება
  • ESP_OK: წარმატებით დასრულდა
  • სხვა: ვერ მოხერხდა

esp_err_t esp_bluedroid_deinit(void)¶

დეინიტაცია და Bluetooth-ის რესურსის გათავისუფლება, ეს ყოველი Bluetooth-ის შემდეგ უნდა გაკეთდეს.

დაბრუნება
  • ESP_OK: წარმატებით დასრულდა
  • სხვა: ვერ მოხერხდა

ჩამოთვლები¶

ჩამოთვლა esp_bluedroid_status_t¶

Bluetooth დასტის სტატუსის ტიპი, რომელიც მიუთითებს, მზად არის თუ არა Bluetooth დასტა.

ღირებულებები:

ESP_BLUEDROID_STATUS_UNINITIALIZED = 0¶

Bluetooth არ არის ინიციალიზებული

ESP_BLUEDROID_STATUS_INITIALIZED¶

Bluetooth ინიციალიზებულია, მაგრამ არ არის ჩართული

ESP_BLUEDROID_STATUS_ENABLED¶

Bluetooth ინიციალიზებული და ჩართულია
