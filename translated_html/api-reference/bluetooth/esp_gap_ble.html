მიმოხილვა¶

ინსტრუქციები

განაცხადის მაგალითი¶

შემოწმება ბლუთუზი/ბლუდროიდი/ბლე ESP-IDF examples-ის საქაღალდეში, რომელიც შეიცავს შემდეგ დემო ვერსიებს და მათ სახელმძღვანელოებს:

  • ეს არის SMP უსაფრთხოების კლიენტის დემო ვერსია და მისი სახელმძღვანელო. ეს დემო ვერსია იწყებს მის უსაფრთხოების პარამეტრებს და მოქმედებს როგორც GATT კლიენტი, რომელსაც შეუძლია უსაფრთხოების მოთხოვნის გაგზავნა თანატოლ მოწყობილობაზე და შემდეგ დაშიფვრის პროცედურის დასრულება.

    • bluetooth/bluedroid/ble/gatt_security_client
    • GATT-ის უსაფრთხოების კლიენტის მაგალითის მიმოხილვა

  • ეს არის SMP უსაფრთხოების სერვერის დემო ვერსია და მისი სახელმძღვანელო. ეს დემო ვერსია იწყებს მის უსაფრთხოების პარამეტრებს და მოქმედებს როგორც GATT სერვერი, რომელსაც შეუძლია გაუგზავნოს დაწყვილების მოთხოვნა თანატოლ მოწყობილობას და შემდეგ დაასრულოს დაშიფვრის პროცედურა.

    • bluetooth/bluedroid/ble/gatt_security_server
    • GATT უსაფრთხოების სერვერის მაგალითის მიმოხილვა
