ფუნქციები¶

კონსტ uint8_t *esp_bt_dev_get_address(void)¶

მიიღეთ Bluetooth მოწყობილობის მისამართი. უნდა გამოიყენოთ „esp_bluedroid_enable“-ის შემდეგ.

დაბრუნება
bluetooth device address (six bytes), or NULL if bluetooth stack is not enabled

esp_err_t esp_bt_dev_set_device_name(კონსტ char *სახელი)¶

Bluetooth მოწყობილობის სახელის დაყენება. ეს ფუნქცია უნდა გამოიძახოს esp_bluedroid_enable() ფუნქციის წარმატებით დასრულების შემდეგ. BR/EDR/LE მოწყობილობის ტიპს უნდა ჰქონდეს ერთი Bluetooth მოწყობილობის სახელი, რომელიც იდენტური უნდა იყოს სახელის აღმოჩენის პროცედურის შესასრულებლად გამოყენებული ფიზიკური არხის მიუხედავად.

დაბრუნება
  • ESP_OK: წარმატებით დასრულდა
  • ESP_ERR_INVALID_ARG: თუ სახელი არის NULL მაჩვენებელი ან ცარიელია, ან სტრიქონის სიგრძე ლიმიტს სცილდება.
  • ESP_ERR_INVALID_STATE: თუ Bluetooth სტეკი ჯერ არ არის ჩართული
  • ESP_FAIL : სხვები
