mDNS სერვისი¶

[中文]

მიმოხილვა¶

mDNS არის მულტიკასტ UDP სერვისი, რომელიც გამოიყენება ლოკალური ქსელის სერვისისა და ჰოსტის აღმოჩენის უზრუნველსაყოფად.

mDNS სტანდარტულად არის დაინსტალირებული ოპერაციულ სისტემებში, ან ხელმისაწვდომია ცალკე პაკეტის სახით. ჩართულია Mac OS ის სტანდარტულად არის დაინსტალირებული და მას ეწოდება BonjourApple-მა გამოუშვა ინსტალატორი Windows რომლის პოვნაც შესაძლებელია Apple-ის მხარდაჭერის გვერდზეჩართულია Linux, mDNS მოწოდებულია ავაჰი და, როგორც წესი, ის სტანდარტულად არის დაინსტალირებული.

mDNS თვისებები¶

  • hostname: მასპინძლის სახელი, რომელზეც მოწყობილობა უპასუხებს. თუ არ არის დაყენებული, hostname ინტერფეისიდან წაიკითხავენ. მაგალითი: my-esp32 გადაწყვეტს my-esp32.local
  • default_instance: თქვენი მოწყობილობისთვის მოსახერხებელი სახელი, მაგალითად Jhon's ESP32 Thingთუ არ არის დაყენებული, hostname გამოყენებული იქნება.

STA ინტერფეისისთვის mDNS-ის გაშვებისა და დაყენების მაგალითის მეთოდი hostname და default_instance:

void start_mdns_service()
{
    //initialize mDNS service
    esp_err_t err = mdns_init();
    if (err) {
        printf("MDNS Init failed: %d\n", err);
        return;
    }

    //set hostname
    mdns_hostname_set("my-esp32");
    //set default instance
    mdns_instance_name_set("Jhon's ESP32 Thing");
}

mDNS სერვისები¶

mDNS-ს შეუძლია თქვენი მოწყობილობის მიერ შემოთავაზებული ქსელური სერვისების შესახებ ინფორმაციის რეკლამირება. თითოეული სერვისი განისაზღვრება რამდენიმე თვისებით.

  • instance_name: თქვენი სერვისის მეგობრული სახელი, მაგალითად Jhon's ESP32 Web Serverთუ არ არის განსაზღვრული, default_instance გამოყენებული იქნება.
  • service_type: (აუცილებელია) სერვისის ტიპი, რომელსაც წინ უძღვის ქვედა ხაზი. შეგიძლიათ იპოვოთ რამდენიმე გავრცელებული ტიპი. აქ.
  • proto: (აუცილებელია) პროტოკოლი, რომელზეც სერვისი მუშაობს, წინ უძღვის ქვედახაზი. მაგალითი: _tcp ან _udp
  • port: (აუცილებელია) ქსელის პორტი, რომელზეც სერვისი მუშაობს
  • txt: {var, val} სტრიქონების მასივი, რომელიც გამოიყენება თქვენი სერვისის თვისებების დასადგენად

რამდენიმე სერვისისა და სხვადასხვა თვისების დამატების მაგალითი მეთოდი:

void add_mdns_services()
{
    //add our services
    mdns_service_add(NULL, "_http", "_tcp", 80, NULL, 0);
    mdns_service_add(NULL, "_arduino", "_tcp", 3232, NULL, 0);
    mdns_service_add(NULL, "_myservice", "_udp", 1234, NULL, 0);

    //NOTE: services must be added before their properties can be set
    //use custom instance for the web server
    mdns_service_instance_name_set("_http", "_tcp", "Jhon's ESP32 Web Server");

    mdns_txt_item_t serviceTxtData[3] = {
        {"board","esp32"},
        {"u","user"},
        {"p","password"}
    };
    //set txt data for service (will free and replace current data)
    mdns_service_txt_set("_http", "_tcp", serviceTxtData, 3);

    //change service port
    mdns_service_port_set("_myservice", "_udp", 4321);
}

mDNS მოთხოვნა¶

mDNS უზრუნველყოფს მეთოდებს სერვისების მოსაძებნად და მასპინძლის IP/IPv6 მისამართების ამოსახსნელად.

სერვისების შედეგები ბრუნდება დაკავშირებული სიის სახით mdns_result_t ობიექტები.

ჰოსტის IP მისამართების გადაჭრის მაგალითი მეთოდი:

void resolve_mdns_host(const char * host_name)
{
    printf("Query A: %s.local", host_name);

    struct ip4_addr addr;
    addr.addr = 0;

    esp_err_t err = mdns_query_a(host_name, 2000,  &addr);
    if(err){
        if(err == ESP_ERR_NOT_FOUND){
            printf("Host was not found!");
            return;
        }
        printf("Query Failed");
        return;
    }

    printf(IPSTR, IP2STR(&addr));
}

ადგილობრივი სერვისების გადაჭრის მაგალითი მეთოდი:

static const char * if_str[] = {"STA", "AP", "ETH", "MAX"};
static const char * ip_protocol_str[] = {"V4", "V6", "MAX"};

void mdns_print_results(mdns_result_t * results){
    mdns_result_t * r = results;
    mdns_ip_addr_t * a = NULL;
    int i = 1, t;
    while(r){
        printf("%d: Interface: %s, Type: %s\n", i++, if_str[r->tcpip_if], ip_protocol_str[r->ip_protocol]);
        if(r->instance_name){
            printf("  PTR : %s\n", r->instance_name);
        }
        if(r->hostname){
            printf("  SRV : %s.local:%u\n", r->hostname, r->port);
        }
        if(r->txt_count){
            printf("  TXT : [%u] ", r->txt_count);
            for(t=0; t<r->txt_count; t++){
                printf("%s=%s; ", r->txt[t].key, r->txt[t].value);
            }
            printf("\n");
        }
        a = r->addr;
        while(a){
            if(a->addr.type == IPADDR_TYPE_V6){
                printf("  AAAA: " IPV6STR "\n", IPV62STR(a->addr.u_addr.ip6));
            } else {
                printf("  A   : " IPSTR "\n", IP2STR(&(a->addr.u_addr.ip4)));
            }
            a = a->next;
        }
        r = r->next;
    }

}

void find_mdns_service(const char * service_name, const char * proto)
{
    ESP_LOGI(TAG, "Query PTR: %s.%s.local", service_name, proto);

    mdns_result_t * results = NULL;
    esp_err_t err = mdns_query_ptr(service_name, proto, 3000, 20,  &results);
    if(err){
        ESP_LOGE(TAG, "Query Failed");
        return;
    }
    if(!results){
        ESP_LOGW(TAG, "No results found!");
        return;
    }

    mdns_print_results(results);
    mdns_query_results_free(results);
}

ზემოთ მოცემული მეთოდების გამოყენების მაგალითი:

void my_app_some_method(){
    //search for esp32-mdns.local
    resolve_mdns_host("esp32-mdns");

    //search for HTTP servers
    find_mdns_service("_http", "_tcp");
    //or file servers
    find_mdns_service("_smb", "_tcp"); //windows sharing
    find_mdns_service("_afpovertcp", "_tcp"); //apple sharing
    find_mdns_service("_nfs", "_tcp"); //NFS server
    find_mdns_service("_ftp", "_tcp"); //FTP server
    //or networked printer
    find_mdns_service("_printer", "_tcp");
    find_mdns_service("_ipp", "_tcp");
}

განაცხადის მაგალითი¶

mDNS სერვერის/სკანერის მაგალითი: პროტოკოლები/მდნმ.

სათაურის ფაილი¶

ფუნქციები¶

esp_err_t mdns_init(void)¶

mDNS-ის ინიციალიზაცია მოცემულ ინტერფეისზე.

დაბრუნება
  • ESP_OK წარმატების შემთხვევაში
  • ESP_ERR_INVALID_STATE, როდესაც მოვლენების დამმუშავებლის რეგისტრაცია ვერ მოხერხდა
  • ESP_ERR_NO_MEM მეხსიერების შეცდომაზე
  • ESP_FAIL, როდესაც mdns დავალების დაწყება ვერ მოხერხდა

void mdns_free(void)¶

mDNS სერვერის გაჩერება და გათავისუფლება.

esp_err_t mdns_hostname_set(კონსტ char *ჰოსტის სახელი)¶

სერვისების რეკლამირებისთვის დააყენეთ mDNS სერვერის საჭირო ჰოსტის სახელი.

დაბრუნება
  • ESP_OK წარმატებით დასრულდა
  • ESP_ERR_INVALID_ARG პარამეტრის შეცდომა
  • ESP_ERR_NO_MEM მეხსიერების შეცდომა
