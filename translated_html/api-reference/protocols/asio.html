ASIO პორტი¶

მიმოხილვა¶

Asio არის მრავალპლატფორმიანი C++ ბიბლიოთეკა, იხილეთ https://think-async.comის თანამედროვე C++ მიდგომის გამოყენებით თანმიმდევრულ ასინქრონულ მოდელს გვთავაზობს.

ASIO დოკუმენტაცია¶

გთხოვთ, იხილოთ asio-ს ორიგინალი დოკუმენტაცია შემდეგ ბმულზე: https://think-async.com/Asio/DocumentationAsio ასევე გთავაზობთ რამდენიმე მაგალითს, რომელთა პოვნაც შესაძლებელია ამ ვებსაიტზე, დოკუმენტაციის/მაგალითების განყოფილებაში.

მხარდაჭერილი ფუნქციები¶

ESP პლატფორმის პორტი ამჟამად მხოლოდ ქსელის ასინქრონული სოკეტის ოპერაციებს უჭერს მხარს; არ უჭერს მხარს სერიული პორტი და ssl-ს. ESP-სთვის შიდა asio პარამეტრები მოიცავს

  • გამონაკლისები ჩართულია ASIO-ში, თუ ჩართულია menuconfig ში.
  • TYPEID ჩართულია ASIO-ში, თუ ჩართულია menuconfig ში

განაცხადის მაგალითი¶

ESP მაგალითები დაფუძნებულია სტანდარტულ asio-ზე. პროტოკოლები/ასიო:

  • პროტოკოლები/asio/udp_echo_server
  • პროტოკოლები/asio/tcp_echo_server
  • პროტოკოლები/asio/ჩატის_კლიენტი
  • პროტოკოლები/asio/ჩატის_სერვერი
