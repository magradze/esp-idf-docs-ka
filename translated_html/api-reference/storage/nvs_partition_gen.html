NVS დანაყოფების გენერატორის პროგრამა¶

[中文]

შესავალი¶

კომუნალური nvs_ ფლეშ მეხსიერება /nvs_partition_generator/nvs_partition_gen.py ქმნის ორობით ფაილს CSV ფაილში მოცემული გასაღები-მნიშვნელობის წყვილების საფუძველზე. ორობითი ფაილი თავსებადია NVS არქიტექტურასთან, რომელიც განსაზღვრულია არასტაბილური საცავიეს პროგრამა იდეალურად შეეფერება ორობითი ბლობის გენერირებას, რომელიც შეიცავს ODM/OEM-ისთვის დამახასიათებელ მონაცემებს და რომლის გარედან ფლეშ მეხსიერება შესაძლებელია მოწყობილობის წარმოების დროს. ეს საშუალებას აძლევს მწარმოებლებს შექმნან ერთი და იგივე აპლიკაციის მრავალი პროგრამული უზრუნველყოფა თითოეული მოწყობილობისთვის მორგებული პარამეტრებით, როგორიცაა სერიული ნომერი.

წინაპირობები¶

To use this utility in encryption mode, install the following packages:
  • კრიპტოგრაფიის პაკეტი

ყველა საჭირო პაკეტი შედის requirements.txt esp-idf დირექტორიის ძირში.

CSV ფაილის ფორმატი¶

.csv ფაილის თითოეული სტრიქონი უნდა შეიცავდეს 4 პარამეტრს, რომლებიც ერთმანეთისგან მძიმით არის გამოყოფილი. ქვემოთ მოცემულ ცხრილში მოცემულია თითოეული ამ პარამეტრის აღწერა.

არა. პარამეტრი აღწერა შენიშვნები
1 გასაღები მონაცემების გასაღები. მონაცემებზე წვდომა მოგვიანებით შესაძლებელია ამ გასაღების გამოყენებით აპლიკაციიდან.  
2 ტიპი მხარდაჭერილი მნიშვნელობებია file, data და namespace.  
3 კოდირება მხარდაჭერილი მნიშვნელობებია: u8, i8, u16, u32, i32, string, hex2bin, base64 და binaryეს განსაზღვრავს, თუ როგორ კოდირდება ფაქტობრივი მონაცემთა მნიშვნელობები შედეგად მიღებულ ორობით ფაილში. განსხვავება string და binary კოდირება არის ის, რომ string მონაცემები მთავრდება NULL სიმბოლოთი, ხოლო binary მონაცემები არ არის. ამ დროისთვის, ამისთვის file ტიპი, მხოლოდ hex2bin, base64, stringდა binary კოდირება მხარდაჭერილია.
4 ღირებულება მონაცემთა მნიშვნელობა. კოდირებისა და მნიშვნელობის უჯრედების namespace ველის ტიპი ცარიელი უნდა იყოს. კოდირება და მნიშვნელობა namespace ფიქსირებულია და არ არის კონფიგურირებადი. ამ უჯრედებში ნებისმიერი მნიშვნელობა იგნორირებულია.

შენიშვნა

CSV ფაილის პირველი ხაზი ყოველთვის უნდა იყოს სვეტის სათაური და მისი კონფიგურაცია შეუძლებელია.

ქვემოთ მოცემულია ასეთი CSV ფაილის მაგალითი:

key,type,encoding,value     <-- column header
namespace_name,namespace,,  <-- First entry should be of type "namespace"
key1,data,u8,1
key2,file,string,/path/to/file

შენიშვნა

Make sure there are სივრცეების გარეშე:
  • ადრე და შემდეგ &#39;,&#39;
  • CSV ფაილში თითოეული ხაზის ბოლოს

NVS შეყვანისა და სახელთა სივრცის ასოციაცია¶

როდესაც CSV ფაილში სახელთა სივრცის ჩანაწერი გვხვდება, თითოეული შემდეგი ჩანაწერი ამ სახელთა სივრცის ნაწილად ჩაითვლება მანამ, სანამ შემდეგი სახელთა სივრცის ჩანაწერი არ მოიძებნება. ამ ეტაპზე, ყველა შემდეგი ჩანაწერი ახალი სახელთა სივრცის ნაწილად ჩაითვლება.

შენიშვნა

CSV ფაილში პირველი ჩანაწერი ყოველთვის უნდა იყოს namespace შესვლა.

მრავალგვერდიანი Blob მხარდაჭერა¶

ნაგულისხმევად, ბინარული ბლობები დაშვებულია რამდენიმე გვერდზე გადაჭიმული და იწერება განყოფილებაში მითითებული ფორმატით. შესვლის სტრუქტურათუ ძველი ფორმატის გამოყენებას აპირებთ, პროგრამა ამ ფუნქციის გამორთვის ვარიანტს გთავაზობთ.

დაშიფვრის მხარდაჭერა¶

NVS Partition Generator პროგრამა ასევე საშუალებას გაძლევთ შექმნათ დაშიფრული ორობითი ფაილი. პროგრამა იყენებს AES-XTS დაშიფვრას. იხილეთ NVS დაშიფვრა დამატებითი დეტალებისთვის.

გაშიფვრის მხარდაჭერა¶

ეს პროგრამა საშუალებას გაძლევთ გაშიფროთ დაშიფრული NVS ორობითი ფაილი. პროგრამა იყენებს AES-XTS დაშიფვრით დაშიფრულ NVS ორობით ფაილს. იხილეთ NVS დაშიფვრა დამატებითი დეტალებისთვის.

კომუნალური პროგრამის გაშვება¶

გამოყენება:

python nvs_partition_gen.py [-h] {generate,generate-key,encrypt,decrypt} ...

Optional Arguments:
+-----+------------+----------------------------------------------------------------------+
| No. | Parameter  |                           Description                                |
+=====+============+======================================================================+
| 1   | -h, --help |        show this help message and exit                               |
+-----+------------+----------------------------------------------------------------------+

Commands:
        Run nvs_partition_gen.py {command} -h for additional help
+-----+--------------+--------------------------------------------------------------------+
| No. | Parameter    |                           Description                              |
+=====+==============+====================================================================+
| 1   | generate     |      Generate NVS partition                                        |
+-----+--------------+--------------------------------------------------------------------+
| 2   | generate-key |      Generate keys for encryption                                  |
+-----+--------------+--------------------------------------------------------------------+
| 3   |  encrypt     |      Generate NVS encrypted partition                              |
+-----+--------------+--------------------------------------------------------------------+
| 4   |  decrypt     |      Decrypt NVS encrypted partition                               |
+-----+--------------+--------------------------------------------------------------------+

NVS დანაყოფის გენერირებისთვის (ნაგულისხმევი):¶

გამოყენება:

python nvs_partition_gen.py generate [-h] [--version {1,2}] [--outdir OUTDIR]
                                     input output size

Positional Arguments:
+--------------+----------------------------------------------------------------------+
| Parameter    |                           Description                                |
+==============+======================================================================+
| input        |        Path to CSV file to parse                                     |
+--------------+----------------------------------------------------------------------+
| output       |        Path to output NVS binary file                                |
+--------------+----------------------------------------------------------------------+
| size         |        Size of NVS partition in bytes (must be multiple of 4096)     |
+--------------+----------------------------------------------------------------------+


Optional Arguments:
+-----------------+--------------------------------------------------------------------+
| Parameter       |                           Description                              |
+=================+====================================================================+
| -h, --help      |     show this help message and exit                                |
+-----------------+--------------------------------------------------------------------+
| --version {1,2} |     Set multipage blob version.                                    |
|                 |     Version 1 - Multipage blob support disabled.                   |
|                 |     Version 2 - Multipage blob support enabled.                    |
|                 |     Default: Version 2                                             |
|                 |                                                                    |
+-----------------+--------------------------------------------------------------------+
| --outdir OUTDIR |     Output directory to store files created                        |
|                 |     (Default: current directory)                                   |
+-----------------+--------------------------------------------------------------------+

NVS დანაყოფის გენერირებისთვის შეგიძლიათ გაუშვათ პროგრამა ქვემოთ მოცემული ბრძანების გამოყენებით: პროგრამას მოყვება CSV ფაილის ნიმუში:

python nvs_partition_gen.py generate sample_singlepage_blob.csv sample.bin 0x3000

მხოლოდ დაშიფვრის გასაღებების გენერირებისთვის:¶

გამოყენება:

python nvs_partition_gen.py generate-key [-h] [--keyfile KEYFILE]
                                         [--outdir OUTDIR]

Optional Arguments:
+--------------------+----------------------------------------------------------------------+
| Parameter          |                           Description                                |
+====================+======================================================================+
| -h, --help         |      show this help message and exit                                 |
+--------------------+----------------------------------------------------------------------+
| --keyfile KEYFILE  |      Path to output encryption keys file                             |
+--------------------+----------------------------------------------------------------------+
| --outdir OUTDIR    |      Output directory to store files created.                        |
|                    |      (Default: current directory)                                    |
+--------------------+----------------------------------------------------------------------+

ქვემოთ მოცემული ბრძანების გამოყენებით, შეგიძლიათ გაუშვათ პროგრამა მხოლოდ დაშიფვრის გასაღებების გენერირებისთვის:

python nvs_partition_gen.py generate-key

დაშიფრული NVS დანაყოფის გენერირებისთვის:¶

გამოყენება:

python nvs_partition_gen.py encrypt [-h] [--version {1,2}] [--keygen]
                                    [--keyfile KEYFILE] [--inputkey INPUTKEY]
                                    [--outdir OUTDIR]
                                    input output size

Positional Arguments:
+--------------+----------------------------------------------------------------------+
| Parameter    |                           Description                                |
+==============+======================================================================+
| input        |        Path to CSV file to parse                                     |
+--------------+----------------------------------------------------------------------+
| output       |        Path to output NVS binary file                                |
+--------------+----------------------------------------------------------------------+
| size         |        Size of NVS partition in bytes (must be multiple of 4096)     |
+--------------+----------------------------------------------------------------------+


Optional Arguments:
+---------------------+--------------------------------------------------------------------+
| Parameter           |                           Description                              |
+=====================+====================================================================+
| -h, --help          |     show this help message and exit                                |
|                     |                                                                    |
+---------------------+--------------------------------------------------------------------+
| --version {1,2}     |     Set multipage blob version.                                    |
|                     |     Version 1 - Multipage blob support disabled.                   |
|                     |     Version 2 - Multipage blob support enabled.                    |
|                     |     Default: Version 2                                             |
+---------------------+--------------------------------------------------------------------+
| --keygen            |     Generates key for encrypting NVS partition                     |
+---------------------+--------------------------------------------------------------------+
| --keyfile KEYFILE   |     Path to output encryption keys file                            |
+---------------------+--------------------------------------------------------------------+
| --inputkey INPUTKEY |     File having key for encrypting NVS partition                   |
+---------------------+--------------------------------------------------------------------+
| --outdir OUTDIR     |     Output directory to store files created                        |
|                     |     (Default: current directory)                                   |
+---------------------+--------------------------------------------------------------------+

NVS დანაყოფის დაშიფვრისთვის შეგიძლიათ გაუშვათ პროგრამა ქვემოთ მოცემული ბრძანების გამოყენებით: პროგრამას მოყვება CSV ფაილის ნიმუში:

  • დაშიფვრა უტილიტასთვის დაშიფვრის გასაღებების გენერირების ნებართვით:

    python nvs_partition_gen.py encrypt sample_singlepage_blob.csv sample_encr.bin 0x3000 --keygen

შენიშვნა

შემდეგი ფორმატის დაშიფვრის გასაღები <outdir>/keys/keys-<timestamp>.bin იქმნება.

  • დაშიფვრა უტილიტას დაშიფვრის გასაღებების გენერირებისა და მოწოდებულ ფაილის სახელში შენახვის ნებართვით:

    python nvs_partition_gen.py encrypt sample_singlepage_blob.csv sample_encr.bin 0x3000 --keygen --keyfile sample_keys.bin

შენიშვნა

შემდეგი ფორმატის დაშიფვრის გასაღები <outdir>/keys/sample_keys.bin იქმნება.

შენიშვნა

ეს ახლად შექმნილი ფაილი, რომელსაც აქვს დაშიფვრის გასაღებები keys/ დირექტორია თავსებადია NVS key-partition სტრუქტურასთან. იხილეთ NVS გასაღების დანაყოფი დამატებითი დეტალებისთვის.

  • დაშიფვრა დაშიფვრის გასაღებების შეყვანის ორობითი ფაილის სახით მიწოდებით:

    python nvs_partition_gen.py encrypt sample_singlepage_blob.csv sample_encr.bin 0x3000 --inputkey sample_keys.bin

დაშიფრული NVS დანაყოფის გასაშიფრად:¶

გამოყენება:

python nvs_partition_gen.py decrypt [-h] [--outdir OUTDIR] input key output

Positional Arguments:
+--------------+----------------------------------------------------------------------+
| Parameter    |                           Description                                |
+==============+======================================================================+
| input        |        Path to encrypted NVS partition file to parse                 |
+--------------+----------------------------------------------------------------------+
| key          |        Path to file having keys for decryption                       |
+--------------+----------------------------------------------------------------------+
| output       |        Path to output decrypted binary file                          |
+--------------+----------------------------------------------------------------------+


Optional Arguments:
+---------------------+--------------------------------------------------------------------+
| Parameter           |                           Description                              |
+=====================+====================================================================+
| -h, --help          |     show this help message and exit                                |
+---------------------+--------------------------------------------------------------------+
| --outdir OUTDIR     |     Output directory to store files created                        |
|                     |     (Default: current directory)                                   |
+---------------------+--------------------------------------------------------------------+

დაშიფრული NVS დანაყოფის გაშიფვრისთვის შეგიძლიათ გაუშვათ პროგრამა ქვემოთ მოცემული ბრძანების გამოყენებით:

python nvs_partition_gen.py decrypt sample_encr.bin sample_keys.bin sample_decr.bin

  • მრავალგვერდიანი Blob-ის მხარდაჭერა გამორთულია (ვერსია 1)
