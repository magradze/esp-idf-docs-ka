წარმოების კომუნალური მომსახურება¶

[中文]

შესავალი¶

ეს პროგრამა შექმნილია მასობრივი წარმოების მიზნებისთვის ქარხნული NVS დანაყოფების სურათების ინსტანციების შესაქმნელად თითოეული მოწყობილობის მიხედვით. NVS დანაყოფების სურათები იქმნება CSV ფაილებიდან, რომლებიც შეიცავს მომხმარებლის მიერ მოწოდებულ კონფიგურაციებსა და მნიშვნელობებს.

გთხოვთ გაითვალისწინოთ, რომ ეს პროგრამა მხოლოდ ქმნის წარმოების ორობით სურათებს, რომლებიც შემდეგ თქვენს მოწყობილობებზე უნდა ფლეშ მეხსიერება შემდეგი გამოყენებით:

  • esptool.py
  • Flash ჩამოტვირთვის ინსტრუმენტი (ხელმისაწვდომია მხოლოდ Windows-ზე)
  • პირდაპირი ფლეშ მეხსიერება პროგრამირება

წინაპირობები¶

ეს უტილიტა დამოკიდებულია esp-idf-ის NVS დანაყოფის უტილიტაზე.

  • Operating System requirements:
    • Linux / MacOS / Windows (სტანდარტული დისტრიბუციები)
  • The following packages are needed to use this utility:
    • საჭიროა Python-ის ვერსია: 2.7 (მინიმუმ). მისი პოვნა აქ შეგიძლიათ:https://www.python.org/downloads/&gt;

შენიშვნა

Before using this utility, please make sure that:
  • Python-ისკენ მიმავალი გზა ემატება PATH გარემოს ცვლადს.
  • თქვენ დააინსტალირეთ პაკეტები requirement.txt, ფაილი, რომელიც მდებარეობს esp-idf დირექტორიის ძირში.

სამუშაო პროცესი¶

CSV კონფიგურაციის ფაილი¶

ეს ფაილი შეიცავს მოწყობილობის კონფიგურაციას, რომელიც ფლეშ მეხსიერება ed-ში უნდა იყოს დაინსტალირებული.

კონფიგურაციის ფაილში მონაცემებს აქვს შემდეგი ფორმატი ( REPEAT ტეგი არჩევითია):

name1,namespace,    <-- First entry should be of type "namespace"
key1,type1,encoding1
key2,type2,encoding2,REPEAT
name2,namespace,
key3,type3,encoding3
key4,type4,encoding4

შენიშვნა

ამ ფაილში პირველი ხაზი ყოველთვის უნდა იყოს namespace შესვლა.

თითოეულ ხაზს უნდა ჰქონდეს სამი პარამეტრი: key,type,encoding, გამოყოფილი მძიმით. თუ REPEAT თუ ტეგი არსებობს, ამ გასაღების შესაბამისი მნიშვნელობა მთავარ მნიშვნელობის CSV ფაილში ყველა მოწყობილობისთვის იგივე იქნება.

თითოეული პარამეტრის დეტალური აღწერილობისთვის იხილეთ NVS Partition Generator უტილიტის README ფაილი.

ქვემოთ მოცემულია ასეთი კონფიგურაციის ფაილის ნიმუში:

app,namespace,
firmware_key,data,hex2bin
serial_no,data,string,REPEAT
device_no,data,i32

შენიშვნა

Make sure there are სივრცეების გარეშე:
  • ადრე და შემდეგ &#39;,&#39;
  • CSV ფაილში თითოეული ხაზის ბოლოს

მთავარი მნიშვნელობის CSV ფაილი¶

ეს ფაილი შეიცავს ფლეშ მეხსიერება რედაქციით გათვალისწინებული მოწყობილობების დეტალებს. ამ ფაილში თითოეული ხაზი შეესაბამება მოწყობილობის ეგზემპლარს.

მთავარ მნიშვნელობაში CSV ფაილში მონაცემებს შემდეგი ფორმატი აქვს:

key1,key2,key3,.....
value1,value2,value3,....

შენიშვნა

ფაილში პირველი ხაზი ყოველთვის უნდა შეიცავდეს key სახელები. კონფიგურაციის ფაილის ყველა გასაღები აქ უნდა იყოს წარმოდგენილი იგივე თანმიმდევრობაამ ფაილს შეიძლება ჰქონდეს დამატებითი სვეტები (გასაღებები). დამატებითი გასაღებები დამუშავდება როგორც მეტამონაცემები და არ იქნება საბოლოო ორობითი ფაილების ნაწილი.

თითოეული ხაზი უნდა შეიცავდეს value შესაბამისი გასაღებების, გამოყოფილი მძიმით. თუ გასაღებს აქვს REPEAT ტეგი, მისი შესაბამისი მნიშვნელობა უნდა შეყვანილია მხოლოდ მეორე სტრიქონში. შემდეგ სტრიქონებში ამ მნიშვნელობისთვის ჩანაწერი ცარიელი დატოვეთ.

ამ პარამეტრის აღწერა შემდეგია:

value
Data value

მონაცემთა მნიშვნელობა არის გასაღების შესაბამისი მონაცემების მნიშვნელობა.

ქვემოთ მოცემულია მთავარი მნიშვნელობის CSV ფაილის ნიმუში:

id,firmware_key,serial_no,device_no
1,1a2b3c4d5e6faabb,A1,101
2,1a2b3c4d5e6fccdd,,102
3,1a2b3c4d5e6feeff,,103

შენიშვნა

თუ „REPEAT“ ტეგი არსებობს, ახალი მთავარი მნიშვნელობის CSV ფაილი შეიქმნება იმავე საქაღალდეში, სადაც შეყვანილი მთავარი CSV ფაილი იქნება, სადაც მნიშვნელობები ჩასმული იქნება „REPEAT“ ტეგის მქონე გასაღების თითოეულ ხაზზე..

ეს პროგრამა ქმნის შუალედურ CSV ფაილებს, რომლებიც გამოიყენება როგორც შეყვანის მონაცემები NVS დანაყოფის პროგრამისთვის ორობითი ფაილების გენერირებისთვის.

ამ შუალედური CSV ფაილის ფორმატი შემდეგია:

key,type,encoding,value
key,namespace, ,
key1,type1,encoding1,value1
key2,type2,encoding2,value2

შუალედური CSV ფაილის ეგზემპლარი შეიქმნება თითოეული მოწყობილობისთვის ინდივიდუალურად.

კომუნალური პროგრამის გაშვება¶

გამოყენება:

python mfg_gen.py [-h] {generate,generate-key} ...

Optional Arguments:
+-----+------------+----------------------------------------------------------------------+
| No. | Parameter  |                           Description                                |
+=====+============+======================================================================+
| 1   | -h, --help |        show this help message and exit                               |
+-----+------------+----------------------------------------------------------------------+

Commands:
        Run mfg_gen.py {command} -h for additional help
+-----+--------------+--------------------------------------------------------------------+
| No. | Parameter    |                           Description                              |
+=====+==============+====================================================================+
| 1   | generate     |      Generate NVS partition                                        |
+-----+--------------+--------------------------------------------------------------------+
| 2   | generate-key |      Generate keys for encryption                                  |
+-----+--------------+--------------------------------------------------------------------+
თითოეული მოწყობილობისთვის ქარხნული სურათების გენერირებისთვის (ნაგულისხმევი):

გამოყენება:

python mfg_gen.py generate [-h] [--fileid FILEID] [--version {1,2}] [--keygen]
                                [--keyfile KEYFILE] [--inputkey INPUTKEY]
                                [--outdir OUTDIR]
                                conf values prefix size

Positional Arguments:
+--------------+----------------------------------------------------------------------+
| Parameter    |                           Description                                |
+==============+======================================================================+
| conf         |        Path to configuration csv file to parse                       |
+--------------+----------------------------------------------------------------------+
| values       |        Path to values csv file to parse                              |
+--------------+----------------------------------------------------------------------+
| prefix       |        Unique name for each output filename prefix                   |
+-----+--------------+----------------------------------------------------------------+
| size         |        Size of NVS partition in bytes                                |
|              |        (must be multiple of 4096)                                    |
+--------------+----------------------------------------------------------------------+

Optional Arguments:
+---------------------+--------------------------------------------------------------------+
| Parameter           |                           Description                              |
+=====================+====================================================================+
| -h, --help          |     show this help message and exit                                |
+---------------------+--------------------------------------------------------------------+
| --fileid FILEID     |     Unique file identifier(any key in values file)                 |
|                     |     for each filename suffix (Default: numeric value(1,2,3...)     |
+---------------------+--------------------------------------------------------------------+
| --version {1,2}     |     Set multipage blob version.                                    |
|                     |     Version 1 - Multipage blob support disabled.                   |
|                     |     Version 2 - Multipage blob support enabled.                    |
|                     |     Default: Version 2                                             |
+---------------------+--------------------------------------------------------------------+
| --keygen            |     Generates key for encrypting NVS partition                     |
+---------------------+--------------------------------------------------------------------+
| --inputkey INPUTKEY |     File having key for encrypting NVS partition                   |
+---------------------+--------------------------------------------------------------------+
| --outdir OUTDIR     |     Output directory to store files created                        |
|                     |     (Default: current directory)                                   |
+---------------------+--------------------------------------------------------------------+

ქვემოთ მოცემული ბრძანების გამოყენებით, შეგიძლიათ გაუშვათ პროგრამა თითოეული მოწყობილობისთვის ქარხნული სურათების გენერირებისთვის. პროგრამას მოყვება CSV ფაილის ნიმუში:

python mfg_gen.py generate samples/sample_config.csmples/sample_values_singlepage_blob.csv Sample 0x3000

მთავარი მნიშვნელობის CSV ფაილს უნდა ჰქონდეს გზა file აკრიფეთ იმ დირექტორიასთან მიმართებაში, საიდანაც ამუშავებთ პროგრამას.

თითოეული მოწყობილობისთვის დაშიფრული ქარხნული სურათების გენერირებისთვის:

ქვემოთ მოცემული ბრძანების გამოყენებით, შეგიძლიათ გაუშვათ პროგრამა თითოეული მოწყობილობისთვის ქარხნული სურათების დაშიფვრისთვის. პროგრამას მოყვება CSV ფაილის ნიმუში:

  • დაშიფვრა უტილიტასთვის დაშიფვრის გასაღებების გენერირების ნებართვით:

    python mfg_gen.py generate samples/sample_config.csv samples/sample_values_singlepage_blob.csv Sample 0x3000 --keygen

შენიშვნა

შემდეგი ფორმატის დაშიფვრის გასაღები <outdir>/keys/keys-<prefix>-<fileid>.bin იქმნება.

შენიშვნა

ეს ახლად შექმნილი ფაილი, რომელსაც აქვს დაშიფვრის გასაღებები keys/ დირექტორია თავსებადია NVS key-partition სტრუქტურასთან. იხილეთ NVS გასაღების დანაყოფი დამატებითი დეტალებისთვის.

  • დაშიფვრა დაშიფვრის გასაღებების შეყვანის ორობითი ფაილის სახით მიწოდებით:

    python mfg_gen.py generate samples/sample_config.csv samples/sample_values_singlepage_blob.csv Sample 0x3000 --inputkey keys/sample_keys.bin
მხოლოდ დაშიფვრის გასაღებების გენერირებისთვის:

გამოყენება:

python mfg_gen.py generate-key [-h] [--keyfile KEYFILE] [--outdir OUTDIR]

Optional Arguments:
+--------------------+----------------------------------------------------------------------+
| Parameter          |                           Description                                |
+====================+======================================================================+
| -h, --help         |      show this help message and exit                                 |
+--------------------+----------------------------------------------------------------------+
| --keyfile KEYFILE  |      Path to output encryption keys file                             |
+--------------------+----------------------------------------------------------------------+
| --outdir OUTDIR    |      Output directory to store files created.                        |
|                    |      (Default: current directory)                                    |
+--------------------+----------------------------------------------------------------------+

ქვემოთ მოცემული ბრძანების გამოყენებით, შეგიძლიათ გაუშვათ პროგრამა მხოლოდ დაშიფვრის გასაღებების გენერირებისთვის:

python mfg_gen.py generate-key

შენიშვნა

შემდეგი ფორმატის დაშიფვრის გასაღები <outdir>/keys/keys-<timestamp>.bin შექმნილია. დროის ნიშნულის ფორმატია: %m-%d_%H-%M.

შენიშვნა

მორგებული სამიზნე ფაილის სახელის მისაწოდებლად გამოიყენეთ –keyfile არგუმენტი.

გენერირებული დაშიფვრის გასაღების ორობითი ფაილის გამოყენება შესაძლებელია თითოეული მოწყობილობისთვის შექმნილი ქარხნული სურათების დაშიფვრისთვის.
