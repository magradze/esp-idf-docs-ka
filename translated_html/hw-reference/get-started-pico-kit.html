ESP32-PICO-KIT V4 / V4.1 დაწყების სახელმძღვანელო¶

[中文]

ეს სახელმძღვანელო გვიჩვენებს, თუ როგორ დაიწყოთ ESP32-PICO-KIT V4 / V4.1 მინი დეველოპერული დაფის გამოყენება. ESP32-PICO-KIT-ის სხვა ვერსიების აღწერისთვის, გთხოვთ, იხილოთ ESP32 აპარატურის ცნობარი.

ეს კონკრეტული აღწერა მოიცავს ESP32-PICO-KIT V4 და V4.1 მოდელებს. განსხვავება მდგომარეობს განახლებულ USB-UART ხიდში, რომელიც V4 ვერსიაში CP2102-დან 1 Mbps-მდე გადაცემის სიჩქარითაა აღჭურვილი და V4.1 ვერსიაში CP2102N-ზეა გადაყვანილი, რომელიც 3 Mbps-მდე გადაცემის სიჩქარითაა აღჭურვილი.

რა გჭირდებათ¶

  • ESP32-PICO-KIT მინი დეველოპერის დაფა
  • USB 2.0 A-დან მიკრო B-მდე კაბელი
  • კომპიუტერი, რომელიც მუშაობს Windows-ზე, Linux-ზე ან macOS-ზე

შეგიძლიათ გამოტოვოთ შესავალი ნაწილები და პირდაპირ გადახვიდეთ განყოფილებაზე აპლიკაციის შემუშავების დაწყება.

მიმოხილვა¶

ESP32-PICO-KIT არის ESP32-ზე დაფუძნებული მინი დეველოპერის დაფა, რომელიც წარმოებულია ესპრესო.

ამ დაფის ბირთვია ESP32-PICO-D4 - სისტემა-პაკეტში (SiP) მოდული, რომელსაც აქვს Wi-Fi და Bluetooth სრული ფუნქციონალურობა. სხვა ESP32 მოდულებთან შედარებით, ESP32-PICO-D4 აერთიანებს შემდეგ პერიფერიულ კომპონენტი ებს ერთ პაკეტში, რომლებიც სხვა შემთხვევაში ცალკე უნდა დაინსტალირდეს:

  • 40 MHz კრისტალური ოსცილატორი
  • 4 მბ ფლეშ მეხსიერება
  • ფილტრის კონდენსატორები
  • RF შესაბამისი ბმულები

ეს დაყენება ამცირებს დამატებითი გარე კომპონენტი 19 ხარჯებს, ასევე აწყობისა და ტესტირების ხარჯებს და ასევე ზრდის პროდუქტის საერთო გამოყენებადობას.

დეველოპერის დაფას აქვს USB-UART ხიდის სქემა, რომელიც დეველოპერებს საშუალებას აძლევს, დაფა კომპიუტერის USB პორტს დააკავშირონ ფლეშ მეხსიერება და გამართვა დაკავშირებისთვის.

ESP32-PICO-D4-ზე არსებული ყველა IO სიგნალი და სისტემის კვება გადის დეველოპერული დაფის ორივე მხარეს განლაგებული 20 x 0.1” ზომის ჰედერის ბალიშების ორ რიგში, მარტივი წვდომისთვის. Dupont-ის მავთულებთან თავსებადობისთვის, 2 x 17 ზომის ჰედერის ბალიშები დატვირთულია მამრობითი სქესის პინების ორი რიგით. ანტენის გვერდით დარჩენილი 2 x 3 ზომის ჰედერის ბალიშები არ არის დატვირთული. საჭიროების შემთხვევაში, ეს ბალიშები შეიძლება მოგვიანებით შეივსოს მომხმარებლის მიერ.

შენიშვნა

  1. 2 x 3 ტიპის პანელები, რომლებიც არ არის შევსებული პინების ჰედერებით, დაკავშირებულია ESP32-PICO-D4 SiP მოდულში ჩაშენებულ ფლეშ მეხსიერება მეხსიერებასთან. დამატებითი ინფორმაციისთვის იხილეთ მოდულის მონაცემთა ცხრილი. დაკავშირებული დოკუმენტები.
  2. ESP32-PICO-KIT სტანდარტულად მამრობითი სათაურებით მოდის.

ფუნქციონალურობის მიმოხილვა¶

ქვემოთ მოცემულ ბლოკ-დიაგრამაზე ნაჩვენებია ESP32-PICO-KIT-ის ძირითადი კომპონენტი 19 და მათი ურთიერთკავშირი.

ESP32-PICO-KIT ბლოკ-სქემა

ფუნქციური აღწერა¶

ქვემოთ მოცემულ ფიგურასა და ცხრილში აღწერილია ESP32-PICO-KIT დაფის ძირითადი კომპონენტი 19, ინტერფეისები და მართვის საშუალებები.

ESP32-PICO-KIT დაფის განლაგება

ქვემოთ მოცემულია ფიგურაზე აღნიშნული ელემენტების აღწერა, ზედა მარცხენა კუთხიდან დაწყებული საათის ისრის მიმართულებით.

ძირითადი კომპონენტი აღწერა
ESP32-PICO-D4 სტანდარტული ESP32-PICO-D4 მოდული მიდუღებულია ESP32-PICO-KIT დაფაზე. სრული ESP32 სისტემა ჩიპზე (ESP32 SoC) ინტეგრირებულია SiP მოდულში, რაც EN სიგნალების გამართული ფუნქციონირებისთვის მხოლოდ გარე ანტენას მოითხოვს LC შესატყვისი ქსელით, გამყოფი კონდენსატორებს და გამწევ რეზისტორს.
LDO 5V-დან 3.3V-მდე დაბალი ვარდნის ძაბვის რეგულატორი (LDO).
USB-UART ხიდი ერთჩიპიანი USB-UART ხიდი: V4-ში CP2102 უზრუნველყოფს 1 Mbps-მდე გადაცემის სიჩქარეს, ხოლო V4.1-ში CP2102N გთავაზობთ 3 Mbps-მდე გადაცემის სიჩქარეს.
მიკრო USB პორტი USB ინტერფეისი. დაფის კვების წყარო, ასევე კომპიუტერსა და დაფას შორის საკომუნიკაციო ინტერფეისი.
5 ვოლტიანი ჩართვის LED ინდიკატორი ეს წითელი LED ინთება, როდესაც დაფას კვება მიეწოდება. დეტალებისთვის იხილეთ სქემები დაკავშირებული დოკუმენტები.
შეყვანა/გამოსვლა ESP32-PICO-D4-ის ყველა პინი დაყოფილია პინების ჰედერებად. თქვენ შეგიძლიათ დაპროგრამოთ ESP32 მრავალი ფუნქციის ჩასართავად, როგორიცაა PWM, ADC, DAC, I2C, I2S, SPI და ა.შ. დეტალებისთვის იხილეთ განყოფილება პინის აღწერილობები.
ჩატვირთვის ღილაკი ჩამოტვირთვის ღილაკი. დაჭერისას დააჭირეთ. ჩექმა და შემდეგ დაჭერით ინგლისური იწყებს Firmware-ის ჩამოტვირთვის რეჟიმს პროგრამული უზრუნველყოფა სერიული პორტი ჩამოსატვირთად.
EN ღილაკი გადატვირთვის ღილაკი.

კვების წყაროს პარამეტრები¶

დაფისთვის ენერგიით მომარაგების სამი ურთიერთგამომრიცხავი გზა არსებობს:

  • მიკრო USB პორტი, სტანდარტული კვების წყარო
  • 5V / GND ჰედერის პინები
  • 3V3 / GND ჰედერის პინები

გაფრთხილება

ელექტროენერგიის მიწოდება უნდა მოხდეს გამოყენებით ზემოთ ჩამოთვლილი ვარიანტებიდან მხოლოდ ერთი, წინააღმდეგ შემთხვევაში შეიძლება დაზიანდეს დაფა და/ან კვების წყარო.

პინის აღწერილობები¶

ქვემოთ მოცემულ ორ ცხრილში მოცემულია სახელი და ფუნქცია დაფის ორივე მხარეს შემავალი/გამომავალი კონექტორების რაოდენობა, იხილეთ ESP32-PICO-KIT დაფის განლაგებაპინების ნუმერაცია და სათაურის სახელები იგივეა, რაც მოცემულ სქემაში. დაკავშირებული დოკუმენტები.

აპლიკაციის შემუშავების დაწყება¶

ESP32-PICO-KIT-ის ჩართვამდე დარწმუნდით, რომ დაფა კარგ მდგომარეობაშია და დაზიანების აშკარა ნიშნები არ აქვს.

ამის შემდეგ, გააგრძელეთ ESP32 აპარატურის ცნობარი, სადაც სექცია ინსტალაცია ეტაპობრივად სწრაფად დაგეხმარებათ განვითარების გარემოს დაყენებაში და შემდეგ ფლეშ მეხსიერება თქვენს დაფაზე პროექტის მაგალითის განთავსებაში.
