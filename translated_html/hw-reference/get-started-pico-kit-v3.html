ESP32-PICO-KIT V3 დაწყების სახელმძღვანელო¶

[中文]

ეს სახელმძღვანელო გვიჩვენებს, თუ როგორ დაიწყოთ ESP32-PICO-KIT V3 მინი დეველოპერული დაფის გამოყენება. ESP32-PICO-KIT-ის სხვა ვერსიების აღწერისთვის, გთხოვთ, იხილოთ ESP32 აპარატურის ცნობარი.

რა გჭირდებათ¶

  • ESP32-PICO-KIT V3 მინი დეველოპერული დაფა
  • USB 2.0 A-დან მიკრო B-მდე კაბელი
  • კომპიუტერი, რომელიც მუშაობს Windows-ზე, Linux-ზე ან macOS-ზე

შეგიძლიათ გამოტოვოთ შესავალი ნაწილები და პირდაპირ გადახვიდეთ განყოფილებაზე აპლიკაციის შემუშავების დაწყება.

მიმოხილვა¶

ESP32-PICO-KIT V3 არის ESP32-ზე დაფუძნებული მინი დეველოპერის დაფა, რომელიც წარმოებულია ესპრესოამ დაფის ბირთვია ESP32-PICO-D4 - სისტემა-პაკეტში (SiP) მოდული.

დეველოპერის დაფას აქვს USB-UART ხიდის სქემა, რომელიც დეველოპერებს საშუალებას აძლევს, დაფა კომპიუტერის USB პორტს დააკავშირონ ფლეშ მეხსიერება და გამართვა დაკავშირებისთვის.

ESP32-PICO-D4-ზე არსებული ყველა IO სიგნალი და სისტემის კვება გადის დეველოპერული დაფის ორივე მხარეს განლაგებული 20 x 0.1” ზომის ჰედერის ბალიშების ორ რიგში, მარტივი წვდომისთვის.

ფუნქციური აღწერა¶

ქვემოთ მოცემულ ფიგურასა და ცხრილში აღწერილია ESP32-PICO-KIT V3 დაფის ძირითადი კომპონენტი 19, ინტერფეისები და მართვის საშუალებები.

ESP32-PICO-KIT V3 დაფის განლაგება

ქვემოთ მოცემულია ფიგურაზე აღნიშნული ელემენტების აღწერა, ზედა მარცხენა კუთხიდან დაწყებული საათის ისრის მიმართულებით.

ძირითადი კომპონენტი აღწერა
ESP32-PICO-D4 სტანდარტული ESP32-PICO-D4 მოდული მიდუღებულია ESP32-PICO-KIT V3 დაფაზე. სრული ESP32 სისტემა ჩიპზე (ESP32 SoC) ინტეგრირებულია SiP მოდულში, რაც EN სიგნალების გამართული ფუნქციონირებისთვის მხოლოდ გარე ანტენას მოითხოვს LC შესატყვისი ქსელით, განმაცალკევებელი კონდენსატორებს და გამწევ რეზისტორს.
LDO 5V-დან 3.3V-მდე დაბალი ვარდნის ძაბვის რეგულატორი (LDO).
USB-UART ხიდი ერთჩიპიანი USB-UART ხიდი უზრუნველყოფს 1 Mbps-მდე გადაცემის სიჩქარეს.
მიკრო USB პორტი USB ინტერფეისი. დაფის კვების წყარო, ასევე კომპიუტერსა და დაფას შორის საკომუნიკაციო ინტერფეისი.
ჩართვის LED ინდიკატორი ეს წითელი LED ინთება, როდესაც დაფას მიეწოდება ენერგია.
შეყვანა/გამოსვლა ESP32-PICO-D4-ის ყველა პინი დაყოფილია პინების ჰედერებად. თქვენ შეგიძლიათ დაპროგრამოთ ESP32 მრავალი ფუნქციის ჩასართავად, როგორიცაა PWM, ADC, DAC, I2C, I2S, SPI და ა.შ.
ჩატვირთვის ღილაკი ჩამოტვირთვის ღილაკი. დაჭერისას დააჭირეთ. ჩექმა და შემდეგ დაჭერით ინგლისური იწყებს Firmware-ის ჩამოტვირთვის რეჟიმს პროგრამული უზრუნველყოფა სერიული პორტი ჩამოსატვირთად.
EN ღილაკი გადატვირთვის ღილაკი.

აპლიკაციის შემუშავების დაწყება¶

ESP32-PICO-KIT V3-ის ჩართვამდე დარწმუნდით, რომ დაფა კარგ მდგომარეობაშია და დაზიანების აშკარა ნიშნები არ აქვს.

ამის შემდეგ, გააგრძელეთ ESP32 აპარატურის ცნობარი, სადაც სექცია ინსტალაცია ეტაპობრივად სწრაფად დაგეხმარებათ განვითარების გარემოს დაყენებაში და შემდეგ ფლეშ მეხსიერება თქვენს დაფაზე პროექტის მაგალითის განთავსებაში.
